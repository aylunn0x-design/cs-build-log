# CS Build Log

Public build log of my CS journey, projects, lessons, and progress over time.

## Purpose
Use this repo to:
- track weekly output
- reflect on mistakes honestly
- document progress in public
- show consistency and engineering growth

## Entry format
Each log should answer:
1. what I built
2. what broke
3. what I learned
4. what I’m improving next
